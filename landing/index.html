Many agents. One runtime. Separate lives.

Run distinct AI agents for work, clients, side projects, and experiments from one local install — each with its own soul, memory, secrets, skills, workspace, and autonomy level. Nothing leaks between them.

Read the docs GitHub
$ bunx @qmilab/asterism init

What you get

  • One agent's memory never appears in another's.
  • One agent's secrets are unreadable from another.
  • Dialable autonomy — propose, notify, autonomous.
  • Even an autonomous agent pauses for confirmation before anything destructive.
  • Reviewable memory: every memory an agent forms is typed, scoped, and yours to approve.

See the guarantees proven end to end in the five-claims walkthrough.

Pairs with Lodestar

Asterism runs your agents and keeps them apart; Lodestar is the layer that makes each one trustworthy — what it knows, believes, and is allowed to do.
