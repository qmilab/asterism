# @qmilab/asterism-core

The engine behind [Asterism](https://github.com/qmilab/asterism): agent identities, per-agent persistence, memory, secrets, trust levels, and the append-only event log.

This is an internal building block. To use Asterism, install the umbrella package: [`@qmilab/asterism`](https://www.npmjs.com/package/@qmilab/asterism).
