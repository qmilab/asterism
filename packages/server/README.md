# @qmilab/asterism-server

Local HTTP endpoint for [Asterism](https://github.com/qmilab/asterism) agents: start runs, list runs, and read an agent's event log over `localhost`.

This is an internal building block. To use Asterism, install the umbrella package: [`@qmilab/asterism`](https://www.npmjs.com/package/@qmilab/asterism).
