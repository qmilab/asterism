# @qmilab/asterism-adapter-pi

Runs [Asterism](https://github.com/qmilab/asterism) agents on the [Pi](https://www.npmjs.com/package/@earendil-works/pi-agent-core) agent runtime.

This is an internal building block. To use Asterism, install the umbrella package: [`@qmilab/asterism`](https://www.npmjs.com/package/@qmilab/asterism).
